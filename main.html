GIT


Git merupakan salah VCS (Version Control System) yang diciptakan oleh Linus Torvalds, yang mana dulunya digunakan untuk pengembangan kernel Linux. Perbedaan dari git dengan version control lain adalah setiap kode yang ditulis oleh developer akan disimpan dalam riwayat disetiap perubahannya
